# Alpaca learning platform

# Functional requirments
claude code integrated trading.
Full read and write access to alpaca
crontab or similar automation that will run every day after market close 
learn from past behavior, journal into data structure that wont blow out context window or tokens
single alpaca account, strategy can be adjusted day to day.
web dash running locally for visibility on strategy and reasoning
goal is to prove whether agentic trading can find an edge

# Non functional 
Dockerized environment
small database to journal learning data structure
docker compose for deploy consistency
